# 🔐 Sistema de Autenticação e Cadastro com Flask e SQLite

Aplicação web desenvolvida para gerenciar cadastro e login de usuários, com foco na integração entre Front-end (HTML5/CSS3) e Back-end (Python/Flask) utilizando banco de dados relacional.

---

## 🛠️ Tecnologias Utilizadas

* **Back-end:** Python, Flask
* **Banco de Dados:** SQLite3
* **Front-end:** HTML5, CSS3

---

## ⚙️ Funcionalidades

- [x] Cadastro de novos usuários (Nome, E-mail e Senha)
- [x] Validação e autenticação no login
- [x] Redirecionamento dinâmico de rotas
- [x] Manipulação do banco de dados SQLite (`CREATE TABLE`, `INSERT`, `SELECT`)
- [x] Prevenção contra SQL Injection utilizando parâmetros preparados (`?`)
