# 🔐 Sistema de Autenticação e Cadastro com Flask e SQLite

Aplicação web desenvolvida para gerenciar cadastro e login de usuários, com foco na integração entre Front-end (HTML/CSS) e Back-end (Python/Flask) utilizando banco de dados relacional.

---

## 🛠️ Tecnologias Utilizadas

* **Back-end:** Python, Flask
* **Banco de Dados:** SQLite
* **Front-end:** HTML, CSS

---

## ⚙️ Funcionalidades

- [x] Cadastro de novos usuários (Nome, E-mail e Senha)
- [x] Validação e autenticação no login
- [x] Redirecionamento dinâmico de rotas
- [x] Manipulação do banco de dados SQLite (`CREATE TABLE`, `INSERT`, `SELECT`)
- [x] Prevenção contra SQL Injection utilizando parâmetros preparados (`?`)

---

## 🚀 Como Executar o Projeto Localmente

1. Clone o repositório:
```bash
git clone https://github.com/Felipe01122/login.git
```

2. Instale as dependências necessárias:
```bash
pip install flask
```

3. Execute a aplicação:
```bash
python app.py
```
Acesse no seu navegador: http://127.0.0.1:5000
